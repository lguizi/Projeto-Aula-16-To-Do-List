# Projeto Aula 16 - To Do List

Este projeto foi desenvolvido como atividade da Aula 16.

## Descrição

Aplicação web simples de lista de tarefas (To Do List).
O usuário pode adicionar tarefas, marcar como concluídas e remover tarefas.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## Funcionalidades

* Adicionar tarefas
* Marcar tarefas como concluídas
* Remover tarefas da lista

## Como usar

1. Abra o arquivo **index.html** no navegador.
2. Digite uma tarefa no campo de texto.
3. Clique no botão **Adicionar** para incluir a tarefa na lista.
4. Clique na tarefa para marcar como concluída ou remover.
